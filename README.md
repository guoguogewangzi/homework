## 5.3 核心行为
恶意代码的核心行为是定义恶意代码传播依赖的载荷或应用、攻击方式、攻击对象等核心行为的描述信息。恶意代码在具有多个行为的情况下，核心行为命名的优先顺序分别是攻击对象、攻击方式、传播依赖的载荷或应用。
